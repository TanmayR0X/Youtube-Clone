YouTube Clone

A responsive, feature-rich clone of YouTube built with React.js, Tailwind CSS, Redux Toolkit, RTK Query, and the official YouTube Data API. Search and watch videos and enjoy a smooth user experience â€” just like the real thing!


 Features

-  Real-time video search using YouTube Data API
-  Video playback with related video suggestions
-  Fully responsive design for desktop and mobile
-  State management with Redux Toolkit
-  API handling with RTK Query
-  Smooth navigation with React Router
-  Tailwind CSS for fast and modern styling
